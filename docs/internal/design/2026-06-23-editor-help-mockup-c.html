Screen 6 · Woven help, A: the markdown hint at the point of typing
cairn's own opportunity, which no major editor takes. The first time an author types a markdown mark (here ## at the start of a line), a small ghost coachmark appears under that line and teaches by letting them finish the gesture. It is anchored to the caret, never blocks typing, and offers two quiet choices: see the cheat sheet, or dismiss. It fires once, then never again. This is the Superhuman single-coachmark idea, pointed at markdown.

Markdown basics

Markdown is plain text with a few marks for formatting. Type the marks on the left of each row and they turn into the formatting on the right. You never have to memorize them; this sheet is always one click away.

A big heading# Title
Bold text**bold**
A section heading## Section
Italic text*italic*
A smaller heading### Smaller
A link[text](url)
A bullet list- item
A quote> quoted
A numbered list1. item
A horizontal rule---
The toolbar above the editor inserts any of these for you. Typing the marks does the same thing.
