Getting started

Your first steps

From a blank site to your first published post

Three small steps. Each one checks itself off as you do it, so the list always shows where you are.

1 of 3 done
  1. Write your first post You have a post started. Open it any time to keep going. Done Open it
  2. Publish it Publishing puts your post on the live site. You can change it and publish again any time. not done Publish a post
  3. Create a page A page is for the parts that stay put, like About or Contact. A post is for news and updates. not done Add a page
These steps follow what is really on your site.
