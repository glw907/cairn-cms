Hi, I am the person behind this site, a small blog built with AstroPaper, ported here as a
cairn theme.

## Tech Stack

This site is built with cairn, a magic-link, GitHub-committing CMS for SvelteKit and Cloudflare.
The theme itself, the header, footer, home, archive, tags, and reading surface, was ported from
[AstroPaper](https://github.com/satnaing/astro-paper), an open-source Astro blog theme by Sat
Naing, released under the MIT license. See this site's README for the full attribution.

## Features

This port carries forward the parts of AstroPaper that make it a good reading surface:

- SEO-friendly, with a canonical URL, an RSS feed, and per-page metadata
- Light and dark mode, with no flash of the wrong theme on load
- A single, capped reading measure that stays comfortable at every width
- Archives, tags, and a client-side search over every post
- Highly customizable, since every color and size in the theme is one token away
