The Tailwind Typography plugin turns a single `prose` class into a full set of sensible reading
defaults: heading rhythm, list markers, code styling, and blockquote treatment, all derived from
the theme's own color and spacing tokens rather than hard-coded values.

cairn's own `prose.css` (part of the engine's chassis layer) follows the same idea: every prose
element reads a design token, so a theme's re-skin carries the reading surface forward at zero
extra cost.
