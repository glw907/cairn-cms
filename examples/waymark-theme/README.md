# Waymark

The opt-in identity layer that turns Wayfinder, cairn's neutral default public theme, back into
Waymark: a display serif, a warm-tinted paper, and the three signature prose gestures. `cairn.pub`
runs this layer on top of Wayfinder.

Install `@fontsource-variable/fraunces`, copy `waymark.css` into your site, and add one import line
directly after your `theme.css` import. See
[`docs/guides/make-wayfinder-your-own.md`](../../docs/guides/make-wayfinder-your-own.md) for the
full walkthrough.
